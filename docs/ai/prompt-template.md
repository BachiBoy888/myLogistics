# Task Template for AI Agent

Этот шаблон используется для постановки задач AI-агенту (Kimi/OpenClaw).

---

# Goal

Краткое описание того, что нужно сделать.

Пример:

"Добавить поле quantity (places) в PL и использовать его в расчете стоимости."

---

# Context

Контекст задачи:

- зачем это нужно
- где используется
- какие части системы связаны

---

# Scope

Что входит в задачу.

Например:

- изменить backend модель
- добавить migration
- обновить API
- обновить UI

---

# Out of Scope

Что **нельзя менять** в этой задаче.

Например:

- не изменять существующие API
- не менять структуру базы
- не делать рефакторинг других модулей

---

# Acceptance Criteria

Условия успешного выполнения задачи.

Пример:

- поле quantity отображается в PL
- данные сохраняются в базе
- расчет стоимости использует quantity

---

# Constraints

AI агент должен соблюдать:

- coding rules из `/docs/ai/coding-rules.md`
- структуру проекта
- существующую бизнес логику

---

# Deliverables

AI агент должен вернуть:

1. branch name
2. commit summary
3. pull request link
4. список изменений
5. как проверить результат
