# Product Context — myLogistics

myLogistics — это веб-сервис для управления логистикой грузов.

Система используется для:

- учета грузов
- объединения грузов в консолидации
- расчета стоимости доставки
- аналитики логистических операций

---

# Основные сущности

## PL (Packing List)

PL — список товаров, отправляемых в рамках одной партии.

PL содержит:

- отправителя
- получателя
- вес
- объем
- стоимость
- валюту
- количество мест (places)

PL может быть добавлен в консолидацию.

---

## Consolidation

Консолидация объединяет несколько PL в одну отправку.

Используется для:

- группировки грузов
- планирования отправок
- отслеживания статусов доставки

Одна консолидация может содержать несколько PL.

---

## Cargo

Cargo — единица груза внутри PL.

Cargo содержит:

- описание
- вес
- объем

---

## Calculator

Калькулятор используется для расчета стоимости логистики.

Расчет может включать:

- стоимость за кг
- стоимость за м³
- дополнительные расходы

---

## Analytics

Раздел аналитики показывает:

- количество PL
- динамику грузов
- вес грузов
- статистику операций

---

# Важные бизнес-правила

1. Один PL может принадлежать только одной консолидации.
2. Статусы PL должны сохранять историю.
3. Аналитика строится на основе данных PL и консолидаций.
4. Изменения UI не должны ломать бизнес-логику.
